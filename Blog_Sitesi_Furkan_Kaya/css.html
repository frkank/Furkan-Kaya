CSS

CSS'in açılımı Cascading Style Sheet'tir. Html ile hazırlanmış içeriklere stil vermek amacıyla kullanılmaktadır. Bu bağlamda Html sayfasında link yardımıyla Css çağrılmalıdır. Bu işlem için 'link:css' kısayolu ile link çağrılır ve 'href' kısmına css dosyasına verilen dosya adı yazılmalıdır.



CSS ilk olarak akla programlama dili gibiymiş gelse de aslında işaretleme dilidir.



CSS ile stil verilirken öncehangi 'html' ögesine stil verileceği belirlenir yani seçilir. Daha sanrasında açılan scope'un içine istenilen özellik ve değerler girilir. Buradaki öğeler footer, header, section vb. gibi html etiketleri ise direkt olarak isimleriyle seçilir. Tanımlanan class çağıralacaksa class isminin önüne nokta('.') konularak seçilir. Aynı şekilde bu işlem id için yapılırsa, id isminin önüne diez('#') konularak ilgli öge seçilir.



CSS stilleme yapılırken oldukça geniş seçenek sunmaktadır. Arka plan renklendirme, yazı tipi ve boyutu özelliştirme, genişlik, yükseklik, sayfaya yerleşim gibi özellikler bu seçeneklerden sıkça kullanılanlarıdır. CSS bu özelliklerin yanı sıra bir çok özelliği bünyesinde barındırır



Bu seçme ve değer değiştirme işlemine biraz örnek verelim;


footer{background-color: red} bu örnekte html ile oluşturulan footer bölümüne arka plan rengi olarak kırmızı değeri girilmiştir. 'Footer' yazarak ilgili ögeyi seçilmiş olunur daha sonrasında açılan scope içindeki 'background-color' ile hangi özelliğe stil verileceği belirlenmiştir. Son olarak 'red' ile özelliğe verilen değer girilir.
